Memory Matching Game

A clean and modern browser-based Memory Matching Game built with HTML, CSS, and JavaScript.
The game features a premium black & gold theme, smooth card-flip animations, multiple board sizes, and automatic best-time tracking.

Features
- Modern black & gold UI with light/dark mode
- Smooth card-flip animations
- Move counter and live timer
- Sound toggle (on/off)
- Best time saved automatically using localStorage
- Multiple board sizes: 4×4, 4×5, 6×6
- Fully responsive layout for all devices

How to Play
1. Choose a board size
2. Click New Game
3. Flip any two cards
4. Match all pairs
5. Try to beat your previous best time

Technologies Used
- HTML5 – Structure
- CSS3 – Layout, animations, theming
- JavaScript (ES6) – Game logic and state management
- LocalStorage – Saves best times and settings
